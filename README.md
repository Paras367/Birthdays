## Project Overview

A password-protected birthday celebration website.

Features:
- Interactive Envelope Opening Animation
- Custom Cursor and Animated Gradient Background
- Click-to-Upload Photo Mosaic and Memory Lane Strip
- Animated Music Player with Web Audio API
- Real-time Confetti Canvas Effects
- Interactive Wish Wall for Birthday Messages
- Live Birthday Countdown Timer
- Cloudflare Worker Password Authentication
- Fully Responsive Mobile-First Design
